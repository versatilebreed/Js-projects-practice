# 📝 Todo List Application

A simple and responsive Todo List web application built using HTML, CSS, and JavaScript.  
This application helps users manage daily tasks efficiently by allowing them to add, complete, and delete tasks.

---

## 🚀 Features

- Add new tasks
- Mark tasks as completed
- Delete tasks from the list
- Responsive and user-friendly interface
- Dynamic task management using JavaScript

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

## ▶️ How to Run the Project

1. Clone the repository

```bash
git clone <repository-link>
```

2. Open the project folder

3. Run the `index.html` file in your browser

---

## 📸 Application Workflow

1. Enter a task in the input field
2. Click the Add button
3. Task gets added to the list
4. Click on a task to mark it as completed
5. Delete tasks when no longer needed
